# FurPHP
## 茸毛 - 周周设计的简易php框架

### 介绍
茸毛是作者周周从自己的项目“幻想动物画廊”的后端业务逻辑总结出的一个php框架。
茸毛被设计为适合在配置有限的服务器上运行，使用极少的资源完成需求，无需进行复杂的配置。

### 启动
- 安装php
- 克隆到本地
- 根目录启动php内置web服务器 ```php -S localhost:80```

### 部署
放置代码到一个生产服务器（如apache/aginx）,支持php，启动web服务即可。
